## Project setup

```
# clone the repo
git clone https://github.com/0xwallet/frontend frontend

# go into app's directory
cd frontend

# install app's dependencies
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Run your tests
```
npm run test
```
### Lints and fixes files
```
npm run lint
```
